Getting started

Download via Bower

PhonePack is available through Bower

bower install phonepack

Donâ€™t use bower?

PhonePack is available via download

Download Phonepack Currently v0.0.1 BETA

Include files

All needed files are in the dist folder. You can include PhonePack files in your html page.

  <link rel="stylesheet" type="text/css" href="phonepack.css">
 <script src="phonepack.js" type="text/javascript" charset="utf-8"></script>

Start the application

  /*Javascript*/
  document.addEventListener('DOMContentLoaded', function() {
  	var phonepack = new PhonePack({
  		fastClick: true // Library for eliminating the 300ms delay between a physical tap and the firing of a click event on mobile browsers.
  	});
  }, false);

  /*jQuery*/
  $(document).ready(function(){
    var phonepack = new PhonePack();
  });

  /*Options*/
  var phonepack = new PhonePack({
    fastClick: true // Library for eliminating the 300ms delay between a physical tap and the firing of a click event on mobile browsers.
  });

Basic template

Use this basic template to get your app started.

<!DOCTYPE html>
<html>
    <head>
      <meta charset="utf-8">
      <meta name="viewport" content="initial-scale=1, maximum-scale=1">
      <title>PhonePack</title>

      <!-- Material Design Icons  -->
      <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
      <!-- Material Design Font  -->
      <link href='https://fonts.googleapis.com/css?family=Open+Sans' rel='stylesheet' type='text/css'>

      <!-- PhonePack CSS  -->
      <link rel="stylesheet" type="text/css" href="phonepack.css">

      <!-- PhonePack JavaScript  -->
      <script src="phonepack.js" type="text/javascript" charset="utf-8"></script>
      <script type="text/javascript">
        document.addEventListener('DOMContentLoaded', function() {
          var phonepack = new PhonePack();
        }, false);
      </script>
    </head>
<body>

  <header class="header bg-lime header-shadow">
    <div class="title">PhonePack</div>
  </header>

  <section class="content has-header">
    <ul class="list">
        <li class="list-item list-item__has-separator">
          <div class="list-content">
            List item 01
            <div class="list-descripton">
              Text Item
            </div>
          </div>
        </li>
      </ul>
    </section>

</body>
</html>
